npm config set registry https://registry.npmjs.org/

0. install backend
    a. Microsoft.EntityFrameworkCore.Sqlite package from nuget gallery to Persistence project
    b. dotnet tool install --global dotnet-ef --version 8.0.3
    c. Microsoft.EntityFrameworkCore.Design packgage from nuget gallery to API project
    d. MediatR by Jimmy Bogard packgage from nuget gallery to Application project
    e. AutoMapper.Extensions.Microsoft.DependencyInjection by Jimmy Bogard packgage from nuget gallery to Application project
    f. FluentValidation.AspNetCore by Jeremy Skinner package from nuget gallery to Application project
0. drop database:
    a. cd Reactivities 
    b. dotnet ef database drop -s API -p Persistence
0. run PostGresQL in docker:
    a. cd Reactivities
    b. docker run --name reactivities_db_dev -e POSTGRES_USER=admin -e POSTGRES_PASSWORD=secret -p 5432:5432 -d postgres:latest
0. Switching Sqlite <-> PostGresQL:
    a. ApplicationServiceExtensions
    b. appsettings.Development.json
0. http <-> https:
    a. API/Properties/launchSettings.json
    b. client-app/vite.config.ts
    c. client-app/.env.development
1. install client
    a. cd Reactivities/client-app
    b. npm i
2. run backend
    a. cd Reactivities/API
    b. dotnet watch --no-load-reload
3. run Client
    a. cd Reactivities/client-app
    b. npm run start
4. build client to production:
    a. cd Reactivities/client-app
    b. npm run build
    c. update sha256 string in program.cs (only if required)
4. run production mode (both backend & client)
    a. cd Reactivities/API
    b. dotnet run
